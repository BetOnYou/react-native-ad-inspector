# react-native-ad-inspector

React Native module for Admob Ad Inspector

## Installation

```sh
npm install react-native-ad-inspector
```

## Usage

```js
import AdInspector from "react-native-ad-inspector";

// ...

const result = await AdInspector.multiply(3, 7);
```
